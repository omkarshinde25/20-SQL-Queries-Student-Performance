# Student Performance Analysis using MySQL

## Project Overview
This is a MySQL mini project focused on analyzing student performance data.
The project is created to practice SQL queries and understand how to work with relational databases using real-world student data.

## Database Details
- Database System: MySQL  
- Database Name: OS  
- Table Name: students  

## Project Objective
To strengthen SQL fundamentals by solving practical queries related to student academic performance.

## Dataset Description
The dataset contains the following student information:
- Student ID  
- Name  
- Department  
- Marks  
- Age  
- City  

## SQL Concepts Used
- Create and use database  
- Create table  
- Insert records  
- SELECT queries  
- WHERE clause  
- ORDER BY  
- GROUP BY  
- HAVING clause  
- Aggregate functions (AVG, MAX, COUNT)  
- Logical operators (AND, OR)  
- LIKE and BETWEEN  
- Subqueries  

## Queries Covered
- Display all student records  
- Show student names and marks  
- Filter students by department  
- Filter students by marks  
- Filter students by city  
- Filter students by age  
- Calculate average marks  
- Find maximum marks  
- Count total students  
- Department-wise average marks  
- Department-wise student count  
- Sort students by marks  
- Sort students by age  
- Multiple condition filtering  
- Marks range filtering  
- Name pattern search  
- Find youngest student  
- Find oldest student in each department  
- Find departments with more than two students  

## Who Can Use This Project
- SQL beginners  
- College students  
- Data analyst freshers  
- Interview preparation  
- Academic mini project  

## How to Use
1. Create the database in MySQL  
2. Create the students table  
3. Insert sample data  
4. Execute queries one by one  
5. Modify queries to explore more insights
