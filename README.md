Student Performance Analysis using MySQL

Project Overview
This project is a MySQL-based student performance analysis mini project.
It is designed to help learners practice SQL queries using a simple and realistic student dataset.

The project focuses on analyzing student academic performance using basic and intermediate SQL concepts.

Database Details
Database System MySQL
Database Name OS
Table Name students

Project Objective
The objective of this project is to develop strong SQL fundamentals and gain hands-on experience in querying and analyzing student data.
